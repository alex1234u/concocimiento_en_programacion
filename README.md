🧅 TOR: ¿Qué es y por qué se habla tanto de él?
Para entender qué es Tor, primero hay que saber cómo funciona un navegador común como Google Chrome, Edge o Firefox.
Estos navegadores te muestran información de páginas indexadas, es decir, páginas que los buscadores conocen y tienen en sus bases de datos.
Imaginá Internet como un iceberg:
•	🧊 La punta es la web que conocés: redes sociales, Wikipedia, YouTube, etc.
•	🌊 Debajo de la superficie está la deep web: páginas no indexadas, como bases de datos académicas, foros privados, archivos médicos, investigaciones o proyectos ocultos a propósito.
Esto no es ilegal, simplemente no está al alcance de Google.
•	⚠️ Pero si seguimos bajando, llegamos a la dark web, la parte más peligrosa del iceberg: ahí sí hay contenido ilegal, malware, estafas, armas, drogas, y todo lo que te imaginás que no debería existir.
________________________________________
⚠️ ¿Es ilegal usar Tor?
No, usar Tor no es ilegal. Es solo un navegador que permite acceder a páginas .onion y navegar con más privacidad.
Pero lo que hagas dentro de Tor sí puede ser ilegal, especialmente si entrás sin saber a la dark web.
Hay trampas, agentes encubiertos, virus, y páginas que pueden dejarte expuesto o incluso atraparte.
Casos como Playpen lo demuestran: mucha gente cayó solo por curiosear donde no debía.
________________________________________
🎯 Reflexión 
No te decimos esto para asustarte, sino para advertirte con respeto.
Tor puede ser usado para el bien: proteger periodistas, activistas, o navegar con privacidad.
Pero si se usa mal, también puede llevarte por caminos peligrosos.
Usalo con cabeza. Usalo con ética. Y sobre todo, saber a qué estás entrando.

🌐 ¿Qué es una IP?
Una IP (Protocolo de Internet) es como la dirección de tu casa, pero en Internet.
Cada vez que te conectás, tu dispositivo (celular, PC, etc.) recibe una dirección IP que le dice a otros servidores quién sos y cómo enviarte información.
Hay dos tipos comunes:
•	IP pública: es la que identifica tu red en Internet (como tu casa en el mundo).
•	IP privada: es la que usan tus dispositivos dentro de tu red (como el cuarto dentro de tu casa).
Con una IP se puede saber desde qué país o ciudad te conectás, y en algunos casos hasta rastrear tu actividad si no estás protegido.

https://www.youtube.com/shorts/kLAPaL2_b58

🔒 ¿Qué es una VPN?
Una VPN (Red Privada Virtual) es como un túnel seguro entre tu dispositivo e Internet.
Sirve para ocultar tu dirección IP real y cifrar (encriptar) todo lo que hacés online.
Cuando usás una VPN:
•	Tu conexión pasa por un servidor externo (que puede estar en otro país).
•	Nadie puede ver qué páginas visitás, ni tu proveedor de Internet, ni hackers.
•	Los sitios web ven la IP del servidor VPN, no la tuya real.
________________________________________
🧠 ¿Para qué sirve una VPN?
•	Navegar de forma más privada y anónima.
•	Evitar bloqueos geográficos (por ejemplo, ver contenido que solo está disponible en otro país).
•	Proteger tus datos en redes públicas (como el Wi-Fi del café).
________________________________________
⚠️ Importante:
Una VPN no te hace 100% anónimo ni invencible, pero mejora mucho tu privacidad y seguridad si la usás bien.

🌐 CONCEPTOS BÁSICOS DEL MUNDO WEB
1. URL
La dirección completa de una página web. Ej: https://youtube.com
2. Dominio
Es el nombre de un sitio web. Ej: google.com — es como la dirección "bonita" de una página.
3. Servidor
Es una computadora que almacena páginas web y responde cuando alguien quiere verlas.
4. Cliente
Es tu navegador o dispositivo que pide esa página al servidor.
5. HTTP / HTTPS
Protocolo de comunicación entre tu navegador y el servidor.
HTTPS es la versión segura (con cifrado).
6. HTML
Lenguaje básico que estructura una página web (como los huesos del cuerpo).
7. CSS
Lenguaje que da estilo (colores, tamaños, posiciones) a lo que ves en la web.
8. JavaScript
Lenguaje que da vida e interactividad a las páginas (botones que se mueven, formularios, etc).
9. Frontend
Todo lo que ves y con lo que interactuás en un sitio (la parte visual).
10. Backend
Lo que está detrás, lo que no ves: bases de datos, servidores, lógica del sistema.
11. Base de datos
Donde se guarda la información de la página: usuarios, contraseñas, publicaciones, etc.
12. Cookies
Archivos pequeños que una web guarda en tu navegador para recordar cosas sobre vos (idioma, carrito de compras, sesión iniciada).
13. Dirección IP
La dirección digital que identifica tu conexión en Internet.
14. VPN
Red privada que oculta tu IP y cifra tu conexión para navegar con más privacidad.
15. Proxy
Un intermediario entre vos e Internet, que puede ocultar tu IP o saltar bloqueos.
16. DNS
Es como la guía telefónica de Internet: convierte nombres como google.com en IPs.
17. Hosting
Servicio que te da un espacio en un servidor para subir tu página web.
18. CMS
"Content Management System" — sistema para crear páginas sin programar (como WordPress).
19. SEO
Técnicas para que tu sitio aparezca mejor posicionado en Google.
20. Phishing
Técnica de estafa donde te hacen creer que estás en una página legítima para robar tus datos.
________________________________________
🚀 Extras útiles si vas más allá:
•	CDN (Red de distribución de contenido): acelera tu sitio entregando copias desde servidores cercanos.
•	API: forma de conectar diferentes servicios o apps entre sí (como pedir datos de clima).
•	Framework: conjunto de herramientas para facilitar la programación (como React, Django, Laravel).
•	Responsivo: diseño web que se adapta a pantallas grandes y chicas.
•	SSL/TLS: cifrado que asegura las conexiones HTTPS.
•	Cache: archivos temporales que se guardan para que las páginas carguen más rápido.
